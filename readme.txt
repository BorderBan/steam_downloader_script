This is my simple script, that downloads items from Steam workshop using SteamCMD.

To use this script You need Python 3 (https://www.python.org/), library Requests for Python (https://pypi.org/project/requests/), SteamCMD (https://developer.valvesoftware.com/wiki/SteamCMD).

To install, just download file "download.py" and put it in SteamCMD directory.

Then run this file. You will see "url: " string. Put in it url of item in workshop.

May not work with some games, but works with RimWorld (verified by me). it uses SteamCMD, so You'l see what's the problem if it has.

Это мой простенький скрипт, который скачивает предметы из мастерской стима через SteamCMD.

Чтобы использовать скрипт, Вам необходим Python 3 (https://www.python.org/), библеотеку Requests для Python (https://pypi.org/project/requests/), SteamCMD (https://developer.valvesoftware.com/wiki/SteamCMD).

Чтобы установить, просто скачайте файл "download.py" и поместите в директорию SteamCMD.

Затем, запустите этот файл. Вы увидите строчку "url: ". Вставьте в неё ссылку на предмет из мастерской.

Может не работать с некоторыми играми, но работает с RimWorld (проверенно мной). Оно использует SteamCMD, так что Вы увидите какая проблема, если она есть.
